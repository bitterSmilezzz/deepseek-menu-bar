# DeepSeek Menu Bar - 产品需求文档

## 1. 产品概述

### 1.1 产品名称
**DeepSeek Menu Bar** - macOS 菜单栏 DeepSeek API 管理插件

### 1.2 产品定位
一款驻留在 macOS 右上角菜单栏的轻量级工具，**解决 DeepSeek 官方 API Key 创建后只显示一次、关闭页面后无法复制的痛点**，提供安全本地存储 + 实时余额用量监控一站式体验。

### 1.3 核心价值（解决什么痛点）
#### 🎯 痛点：DeepSeek 官方新建 API Key 只显示一次
- 用户在 DeepSeek 官网创建 API Key 后，**页面仅展示一次完整密钥**
- 一旦关闭弹窗或刷新页面，密钥被星号隐藏，无法再完整复制
- 如果当时忘记保存，只能删除重建，浪费时间和 API 调用配额
- 多人协作场景下，Key 传递和使用情况难以管理

### 1.4 核心使用场景
- 🔑 **新建 Key 后立即保存** — 在 DeepSeek 官网建好 Key 后，第一时间添加到本工具，永不再丢
- 👁️ **瞄一眼看余额** — 编码时扫一眼菜单栏就知道账户状态
- 📊 **每天看用量** — 查看当日 Token 消耗量，掌握消费情况
- 🔄 **多账号切换** — 快速切换个人号、团队测试号、备用号
- 📰 **资讯动态** — 浏览 DeepSeek 的最新动态

### 1.5 用户画像
- AI 应用开发者
- 深度使用 DeepSeek API 的技术人员
- 需要管理多个 API Key 的团队

## 2. 功能需求

### 2.1 功能总览

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 账户管理 | 添加/编辑/删除 API Key | P0 |
| 账户管理 | 多 Key 一键切换 | P0 |
| 余额展示 | 实时显示账户余额（USD） | P0 |
| 用量监控 | 当日 Token 消耗量 | P0 |
| 工具列表 | DeepSeek 支持的工具一览 | P1 |
| 资讯动态 | DeepSeek 最新新闻搜索 | P1 |
| Pin 置顶 | 面板可置顶在桌面 | P0 |
| 主题切换 | 深色/浅色模式切换 | P0 |
| 主题切换 | 跟随系统自动切换 | P1 |

### 2.2 详细功能描述

#### 2.2.1 菜单栏图标
- 应用启动后在菜单栏右侧显示 DeepSeek 图标
- 无 Dock 图标，不占用 Dock 空间
- 图标可随余额状态变化（低于阈值显示警示色）

#### 2.2.2 弹出面板（点击图标后）
- 点击菜单栏图标，从图标下方弹出面板
- 面板采用毛玻璃背景，大圆角设计
- 点击面板外部区域自动关闭（关闭可 pin 模式除外）
- 面板尺寸：宽度 360px，高度自适应

#### 2.2.3 首页看板（面板默认视图）
```
┌──────────────────────────────────┐
│ [🔒] sk-xxxx...xxxx    [账号▼]  │
│                                  │
│  💰 余额                         │
│  $12.34  ← 大字醒目显示          │
│                                  │
│  📊 今日用量                     │
│  输入: 45,231 tokens             │
│  输出: 12,089 tokens             │
│  总计: 57,320 tokens             │
│                                  │
│  ┌──────────────────────────┐    │
│  │  🔧 可用工具 (5)    〉   │    │
│  │  📰 最新资讯        〉   │    │
│  └──────────────────────────┘    │
│                                  │
│  [📌 置顶]   [🔄 刷新]  [⚙️]   │
└──────────────────────────────────┘
```

#### 2.2.4 Pin 置顶功能
- 点击 📌 按钮可 Pin 住面板
- Pin 住后点击面板外部不会关闭
- 再次点击 📌 或按 Esc 取消置顶
- Pin 住时面板可随意拖动位置

#### 2.2.5 主题切换
- 支持 **暗夜模式**（默认）和 **日照模式** 两种主题
- 暗夜模式：深色背景（#0a0a0f）+ 毛玻璃面板，适合低光环境
- 日照模式：浅色背景（#f5f6fa）+ 白色毛玻璃面板，适合白天使用
- 面板内设置页提供「跟随系统」选项，自动匹配 macOS 深色/浅色外观
- 主题切换动画：0.35s 平滑过渡，所有颜色变量统一管理

#### 2.2.6 二级页面
- **工具列表页**: 展示 DeepSeek 所有可用工具名称和说明
- **资讯列表页**: 搜索并展示 DeepSeek 相关新闻，可点击跳转
- **设置页**: 管理 API Key、主题、刷新间隔等

## 3. 非功能需求

### 3.1 性能要求
- 应用内存占用 < 50MB
- 冷启动时间 < 2 秒
- API 数据自动刷新间隔可配置（默认 5 分钟）
- 面板展开/收起动画流畅（60fps）

### 3.2 安全要求
- API Key 使用 macOS Keychain 加密存储
- 所有网络通信强制 HTTPS
- 不在日志或缓存中明文存储 API Key

### 3.3 兼容性
- macOS 13.0 (Ventura) 及以上
- 支持 Apple Silicon (M1/M2/M3/M4) 和 Intel
- 支持浅色/深色模式

### 3.4 交互规范
- 面板弹出动画：从图标位置向下展开，0.2s ease-out
- 数据刷新加载态：骨架屏，不改变面板尺寸
- 错误状态：面板内显示错误提示 + 重试按钮

## 4. 用户体验流程

```
用户首次使用（新建 Key 后立即保存）:
  在 DeepSeek 官网新建 API Key → 复制完整 Key
  → 启动本工具 → 菜单栏出现图标 → 点击图标
  → 弹出面板提示「添加 API Key」→ 粘贴 Key 并命名
  → Key 安全存入 macOS Keychain → 自动获取余额 → 进入首页看板

日常使用:
  点击菜单栏图标 → 弹出面板查看余额/用量 → 点击外部关闭

切换账号:
  点击顶部账号下拉菜单 → 选择其他 Key → 面板自动刷新数据

Pin 置顶:
  点击 📌 → 面板不消失 → 可拖动到任意位置 → 再次点击 📌 或按 Esc 取消
```

### 4.1 扩展：设置页 API Key 管理流程

```
添加 Key:
  点击「＋添加 API Key」→ 弹出输入框 → 输入名称 + 粘贴完整 Key
  → 点击保存 → Keychain 加密存储 → 自动切换到该 Key → 获取余额

查看 Key:
  Key 在列表中始终以掩码形式显示（sk-xxxx••••••xxxx）
  点击 👁️ 可暂时明文显示（3 秒后自动隐藏）

编辑 Key:
  点击 ✏️ → 可修改名称或重新粘贴 Key → 保存更新

删除 Key:
  点击 🗑️ → 确认删除 → 从 Keychain 中移除
```

## 5. 视觉风格

- **主题**: 暗夜模式 / 日照模式 双主题，支持手动切换
- **跟随系统**: 可设置自动匹配 macOS 外观模式
- **切换动画**: 0.35s CSS 变量驱动平滑过渡
- **背景**: 玻璃拟态毛玻璃效果（backdrop-blur）
- **强调色**: DeepSeek 蓝 (#3B82F6) → 紫 (#8B5CF6) 渐变（双主题统一）
- **字体**: SF Pro（系统原生字体）
- **圆角**: 大圆角 16px
- **图标**: SF Symbols（系统原生图标）
